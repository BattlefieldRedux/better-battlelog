Do you have the "Delayed Contextmenu" option activated ?
All custom BBLog Contextmenus are delayed. That means you must hold down the right mouse button for a while (approx. half of a second) to get the BBLog Contextmenu. If you release the right mouse button immediately than the browser default contextmenu been displayed. That give you the ability to use both menus without the need to deactivate BBLog Contextmenu


You can try to desactivate this option if you are not comfortable with it ?!
