As posted in the feature request I created a plugin for keyboard navigation in assignment and dogtag list.

Github Url:
https://github.com/alexander-schranz/bblog-keyboard-navigation


Plugin Url:
https://getbblog.com/en/plugin/147278/List-Keyboard-Navigation


You can navigate with the Arrow Keys or WASD in the Assignement or Dogtag list. With enter you can equip your dogtags or track the current assignment.
