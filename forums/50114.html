Hi Community, i am Roland, Developer and Admin of Better Battlelog!

BBLog is nearly 1 year old and it has become a bigger thing that i ever had imagined.

Since Version 3.0.0 the Daily User Peak is over 4000 People Online, at the same time, with our BBLog Addon. This is 2x times more than before 3.0.0.

Better Battlelog requires a lot of interaction with the BBLog Server to work properly, for example the Image Upload for Platoons or the PDF Generator for Battlereports.

But i have one big problem. Server Capacity.
The BBLog Server can't handle so much requests anymore.

At the moment, i pay the server from my own pocket. Which is 360€ per Year.

A Server that can handle all required things yet and also in the near future costs 600€ per Year.

I definitely not will pay more money as i currently does from my own pocket.

So the thing that i want to point out with this thread is that i have now 3 options

1. Get a new server that can handle current requests (min. Requirements: Root Dedicated Server, Quad Core, 4GB Ram). I don't buy one.
---
2. Turn off Better Battlelog (what i don't will do)
----
3. Remove Better Battlelog Features that requires the Server (Platoon Images, PDF Generator, etc...) That's what i will do.

Advertisment like Google Ads is no option, the most people doesn't visit the website, they just use BBLog on Battlelog. So there be no income.
The only way to keep all current features is a sponsored Root Server for BBLog.

For all those really want to donate for BBLog, just contact me via private message.

Sorry for that people but there is no other way anymore to handle this massive user interest.
