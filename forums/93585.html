It's pretty easy. :-)

Delete group = Hover over group, a x will appear on the right side, klick it, group deleted
Rename group = No rename function integrated, you have to delete the group and create a new one
OR
you export your BBlog settings, open bblog-settings.json, search the group name, change it, save the file and import it back into BBlog.
