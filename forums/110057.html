If you click the "Add to Radar" button on the server page it'll add an undefined entry in the server radar list. The only way to successfully add a server to the radar is using the context menu (right click -> add to radar).

Also: the link in the server radar list should be "/bf3/servers/show/pc/server-id/" instead of "/bf3/servers/show/server-id/" otherwise you receive a 404 error.
