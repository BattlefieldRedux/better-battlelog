User Tagger:

currently the EA ID/Origin ID/Battlelog ID whatever you want to call it - it doesn't always match up with a players Xbox Live Gamertag ID.

All of these different identities make it difficult to remember who is who.
I know it shows the PC, XBOX, and PS3 Soldiers that are tied to a particular Battlelog ID on all users' Battlelog Profiles. I just think it would be easier to identify who is who on Battlelog if we were able to display our own name for the user that would override their Battlelog ID.

For example:
my Battlelog ID is ViciousTwistt because ViciousTwist was already taken.
my Xbox Live Gamertag is ViciousTwist.
I changed my Battlelog ID to help make it easier for people, but a lot of players don't know how to change their Battlelog ID's or even care to change it.

I want to be able to tag a user's Battlelog ID and rename it whatever I want.
instead of his Battlelog ID being displayed as JohnDoeSoldier11, I want to be able to rename it to whatever his Gamertag was when we played and met on Xbox Live, so I can better remember who he is.

example:
his Battlelog ID is JohnDoeSoldier11.
his Xbox Live Gamertag is Aceyalone.
I want to be able to rename his Battlelog ID, so that it shows up as Aceyalone for me across the entire Battlelog website. I never want to see the name JohnDoeSoldier11 again, because I won't remember who that is.

Thanks,
ViciousTwist
