Hi all!
If you like hunting dogtags this plugin is for you. It adds some features to the dogtags page.
Tip: You can search in the list with
Ctrl+F
;)

= Installation=

Like other BBlog's plugins:
- click on
-BetterBattlelog-

- click on
Plugins
button
- at the panel's bottom, copy/past this link in the field:
Dogtags Enhancer:
https://dl.dropboxusercontent.com/s/jvl8hwdy92z0qz9/dogtagenhancer.js

- press Enter
That's all.

= Features =

- add a new dogtags filter
- display dogtags in list to highlight remaining dogtags (
Ctrl+F
to search the list)
- 2 options : display locked dogtags, normal/extended list
