I don't know if many people knows that there are some tools out on the internet that helps you configure Battlefield 3. I for one didn't know about the tools until today when I by accident happen to stumble upon the Realmware platoon.

Anyhow, this tool is simply great :)
It let's you configure all the Battlefield 3 settings out of game and even includes some hidden features that are hidden within the source code of Battlefield 3, it does all that and some more but still manages to squeeze in some game fixes as well (example: MAV key bindings).

This tool could be good for film makers, it includes the option to always have the HUD on/off and some Field of View (FOV) settings.

The tool I'm talking about is made by Realmware, you can grab it here:
http://bf3.realmware.co.uk/settings-editor/


NOTE 1: You may want to turn off the cloud feature in Origin, unless you want to be botherd with it every time after you've made any changes with the tool.
NOTE 2: This won't get you banned by PunkBuster, look at the FAQ page on their website for more information.

elementofprg
wasakakero wrote:
Best tool ever, someone knows how to config to get max fps without visuals things that i dont need?



To help push the max out of BF3 on my slightly aging system. I went into my Program Files(x86)/Origin Games/Battlefield 3/ folder and made a text file and name it user.cfg

In that text file I have:
GameTime.MaxVariableFps 200.000000
PostProcess.DynamicAOEnable 0
Render.DrawFps 1
Render.DrawScreenInfo 0
Render.PerfOverlayEnable 1
Render.PerfOverlayVisible 0
RenderDevice.Dx11Enable 1
RenderDevice.ForceRenderAheadLimit 0
RenderDevice.TripleBufferingEnable 0
Screenshot.Format png
UI.DrawEnable 1
WorldRender.DxDeferredCsPathEnable 0
WorldRender.MotionBlurEnable 0
WorldRender.SpotLightShadowmapResolution 256
WorldRender.TransparencyShadowmapsEnable 0

That will bump your FPS up with minimal quality loss. If it starts to stutter on you, its most likely RenderDevice.ForceRenderAheadLimit 0 Change 0 to 1,2, or 3 what ever one fixes it.

(P.S. Any command you can put into the command console (~) you can put into the user.cfg and it will make it so you don't have to type it in every time.
