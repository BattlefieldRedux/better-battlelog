This plugin adds small features to many areas of Battlelog. All of the ideas were submitted by GeneralHastati, thanks!

Feedback and translations to other languages will be appreciated.

Features:

Add a Leaderboards dropdown
Remove the secondary menu
Add a Missions menu item
Remove the 'What are Battlepacks?' box
Add mission names to their buttons
Hide Missions from the Multiplayer dropdown
Add a Missions item after Awards in the Soldier dropdown
Add a link to bf4stats.com to the Soldier dropdown
Open the bf4stats.com link in a new tab

Changelog
1.1.1 - added a portuguese translation (thanks to ArnoldLayne)
1.1 - added an option to open the bf4stats.com link in a new tab
1.0 - initial version

You can install this plugin from the plugin gallery:
http://getbblog.com/plugin/126690/General-Enhancements
