Some of you might know this already, but for all the others it's very useful!

Battlefield 3 graphics ain't the best, there is a weird blue layer on top of it, almost everybody is annoyed by it. However, there was a time called Alpha and Beta where the blue layer wasn't there, yet.
Some dev's were looking into the codes and found a possibility to disable that blue layer, so all colors would be more realistic. There are two different "FXAA injector's", they both do they same. It's just which one you prefer the most.

INSTALLATION

Danoc1: 1. Download the file;
2. Unzip the file (with Winrar, Winzip, 7-Zip or whatever);
3. Double click the .exe file;
4. Play the game.

Some Dude:1. Download the file;
2. Unzip the file (with Winrar, Winzip, 7-Zip or whatever);
3. Place all files in your Battlefield 3 directory where the game's bf3.exe is located (most likely C:\Program Files (x86)\Origin Games\Battlefield 3);
4. Play the game.


DE-INSTALLATION

Danoc1: 1. Open your Battlefield 3 directory where the game's bf3.exe is located (most likely C:\Program Files (x86)\Origin Games\Battlefield 3);
2. Delete the following files:
[list] - injFX_Shaders;
- dxgi.dll;
- injFX_Settings;
- shader;
- user.cfg.
3. Done.[/list:u]

Some Dude: 1. Open your Battlefield 3 directory where the game's bf3.exe is located (most likely C:\Program Files (x86)\Origin Games\Battlefield 3);
2. Delete the following files:
[list] - injFX_Shaders;
- dxgi.dll;
- injFX_Settings;
- shader.
3. Done.[/list:u]
