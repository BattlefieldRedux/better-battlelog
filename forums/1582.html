On some few configuration it could happen that chrome looses all addon settings, but currently i don't have a fix for this. The problem is that battlelog delete sometimes the complete dom storage (where the addon settings are saved). For firefox this cannot happen because they use another storage engine.
So, i need to do some tests for this issue.
Tip: You can also save your filters as a simple bookmark. Just create a set, load it and bookmark the url. As a workaround.
