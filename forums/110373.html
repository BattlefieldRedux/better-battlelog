For clarification:

Soldier/User Radar = Warn you (with a message next to the join server button) when the user from your radar is currently on this server. Also there is a additional option to make sound when you viewing a server detail page with activated live scoreboard.

Flame Radar = Just for the board. If a board user, that is on your radar, write a post, than the post will be hidden.

Server Radar = Warn you (with a message next to the join server button) for a server that is on your radar list.
