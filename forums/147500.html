Name: BF4 Live Scoreboard Sorter
Author: I-MrFixIt-I
Version: 0.1 (Beta)

☰ Description:
Normally the scoreboard is sorted descending by the score. But as an admin (even as a normal player) it's sometimes easier sorting a table by your own needs. And this plugin will do excatly this for you :)

☰ Options:
- "Use Live-Scoreboard Sorter" (default: ofn: Toggle the plugin
- "Synchronize both tables" (default: on): Click on one table header and both tables will be sorted by the clicked column
- "Consider clan-tag" (default: off): Sorting by the soldiername (first column) the clan-tag will be considered while sorting the table
