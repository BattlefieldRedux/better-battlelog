I think a lot of people might find this useful:

It would be great if we could set a 320x320 image as the "background" on the emblem editor's canvas, which could then be "traced over" using the editor's shapes and tools.

Obviously the image would not be saved with the emblem, but it would allow more accurate reproduction of source material, and would also allow people to sketch out designs in other graphics software to use as a "stencil" in the emblem editor.

I've already been doing this manually with my browser's developer tools but to be able to do it quickly through BBLog would be awesome.
