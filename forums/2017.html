Hi guys,

I don't want to be merging two themes together, it's an extra work to do. Every update of one theme we would have to update the other one as well.

The easiest way to have the two together, is to simply enable minilog theme in the theme switch and then use the new function - add own theme and use this link:
http://getbblog.com/themes/dark-theme-juress.css


I dont know whether its gonna update itself automatically, but it'll be one version behind every time, since the new version is using a version parameter.
