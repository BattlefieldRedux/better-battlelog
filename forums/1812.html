This is regarding the "Radar" feature.

1) I am not sure "Radar" is a good name for this, it doesn't have an instant relate-able feel to it. Instead you should call it "Blacklist" (Blacklisted Player, Blacklisted Server). It is a very easy word to instantly understand what this feature is for and gives it a clear definition.
I have seen quite a few posts about people that aren't sure what the Radar feature is used for. If it was called Blacklist, that wouldn't be an issue.

2) When you Blacklist a player or a server, allow users to add a message that will be easily displayed when you try to join a server with a blacklist mark on it (or with a player in that server). It would be nice to know why you blacklisted it/them in the first place.
Possibly just make it a hover over (If you hover over the "Blacklisted" mark it will display the message you have written - so it doesn't tale up unnecessary space.
