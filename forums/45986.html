[3.6.x] Remove Elements
http://bblog.p0mp.com/remove-elements/plugin.js


This plugin allows you to remove certain elements from the Battlelog interface. It cleans up the interface and removes elements that people can consider obsolete, intrusive or even dangerous ('stat reset button') Below is a list of things you can enable/disable.

+ Hide stat reset element
+ Hide main Battlelog logo
+ Hide main tiles element
+ Hide main shortcut element
+ Hide pre-order element
+ Hide game activity element
+ Hide entire right menu
+ Clear bottom space (not working)


For more information about the author, contributors and the plugins, visit
http://p0mp.com
