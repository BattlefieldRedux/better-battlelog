I do not know if it still works (encode - decode) but here are the info you want:

Location / Settings:
Untitled.jpg


How it works:
You set a keyword for a specific platoon. When you go to your platoon messages you will see an "encode" button (under the area where you write a new message). You write the message you want and you click "encode". The plugin will encode your message based to the keyword you have set for this platton. Every person that has this plugin and has set the same keyword, will be able to decode your message and see it!

So, only your platoon memebers will be able to see what you are writing (you will set a keyword that all the platoon members will use).

P.S.: I will contact Darkthanos to tell him to improve this plugin. It is very useful.
