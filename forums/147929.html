Name: Friends Highlighter

Version: 1.0

Since the source code for a plugin like this is no longer available I made my own version.

☰ Description:
Highlights your friends in the scoreboard and battlereports

☰ Options:
- "Use Friends Highlighter" (default: on): Toggle the plugin
- "Highlight friends in scoreboard" (default: on): Highlights your friends on the live scoreboard
- "Highlight friends in battlereport" (default: on): Highlights your friends on a battlereport
- "Change color" (default: off): Change the highlighting color (need to be on to take effect, else standard color)
