This plugin is UNDER DEVELOPMENT (it is not fully working).

The idea is to make a plug in which will help the platoons to set up clan wars and recruit new members. A list of platoons (like the server browser) where you will be able to "Chalenge to war" other clans. You will be able to enlist your platoon (if you are a Founder/Leader) in the list and select on which game modes / map / player number you would like to fight. Other clans will be able to challenge you to a war through the plugin (based on their interests) and contact with you.

Being a currently active competitive BF4 player, I think this is an amazing idea, it can actually make the competitive a little bit better. Thanks to the useless DICE, currently you can play competitive matches only by using external sites (such as ESL) or by adding your opponent as a friend (little bit annoying considering the 100 friends limit). So yeah, this plugin can really improve the competitive for BF4, at least when searching for an opponent.

First, I'd suggest you to focus on the basic stuff, having a ladder system with points and everything is awesome but it's not necessary. The main and most important thing it's gonna be the organization of the matches.

So basic stuff are:
- the team name (e.g. the platoon name or maybe allow a custom name too);
- some extra custom info (e.g. specific rules and limitations);
- the game mode (e.g. domination 5vs5, defuse 5vs5, conquest 8vs8);
- the server address (very useful imho);
- optional: the server password (only visible after accepting the match);
- the username of the match submitter (the founder/leader that added the challenge, so if you want you can add him to friends).

Having these stuff is gonna be a good starting point. :)
