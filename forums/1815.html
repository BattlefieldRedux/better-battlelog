I believe you should extend the Kit filters and split the class kits and gun types into 2 different sections.

- The first section will be the Class Type (Assault, Engineer, Support, Recon).
- The second section will be the Weapon Type (Assault Rifle, Carbines, Light Machine Gun etc)

1) You can only choose one in each section (But don't have to select one in each), for example 1-Engineer 2-Carbines. This will help players get rid of some of the trash in each sections (Things like the RPG's and Mortar's that maybe they don't want to see).

2) Not only will it help people clean-up what they see (or what they want to see), it will help people see which type of weapon each gun is, which will help out for assignments (Which guns are Carbines? etc).

3) You COULD hide buttons depending on what class you are looking at (You don't need to see RPG's if you are viewing the Assault class). But it would be nice to be able to look at all SMG's regardless of class.
