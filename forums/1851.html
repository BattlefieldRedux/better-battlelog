This plugin integrates the playing Dice employees in your comcenter. This allows you to join their servers in one click. You can also see if they have a Dice or a Dev team dogtag equipped.
The list of players comes the the Official DICE Platoon and is refreshed every 5 minutes.

https://raw.github.com/azixMcAze/DiceFriends/hunt-dice/DiceFriendsPlugin.js


Note : The dice players int the com center are drag and drop-able like any online friend, but I have no idea if you can invite them to a party or to join a server. Maybe it is better to not try :). But anyway, I will try to prevent that in a future release

As I said in Pun1a's thread, it seems that I am late with this plugin, but the plugin is done and it would be a waste to not share it anyway :)

Versions :
1.1.1-hunt-dice (29 sept) : Added a configuration window to filter the players by platform or by dogtag
1.1.0-hunt-dice (27 sept) : Alternate version getting its data from
http://www.furydivine.net/hunt-dice/
supports console players and DICE Friends dogtags
1.0.3 (11 July) : DICE player list should no longer disappear when friends log in or log out
1.0.2 (5 July) : No longer displays player on ps3 or xbox
1.0.1 (3 July) : Support for languages other than English
1.0 (3 July) : First release

(old version, no hunt-dice :
https://raw.github.com/azixMcAze/DiceFriends/master/DiceFriendsPlugin.js
)
