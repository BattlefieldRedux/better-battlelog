This plugin is inspired by the discontinued Chrome extension BattlelogPlus.
It simply adds the average rank and score, for each team on the selected server, to the scoreboard in the serverbrowser.

BBPlusScreen.png


Link:
https://dl.dropboxusercontent.com/u/35468490/Plugins/sbplus.min.js


Developer Version (not minified):
https://dl.dropboxusercontent.com/u/35468490/Plugins/SBPlus.js

CSS Template:
https://dl.dropboxusercontent.com/u/35468490/Plugins/example.css
