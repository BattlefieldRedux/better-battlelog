[BF4] X-RAY Dark Glass BBLog Theme
by jpfive
Description:
XRAY is a dark glass BF4 battlelog theme that sticks to the basics of clean and minimal design.

- Made for the every-day user
- Easy navigation
- Dark and clean layout
- Active Buttons light up green
- Server Browser simplified and cleaned up
- Clean integrated Google Fonts: Roboto

It's the Small things that make a big difference!

*This theme is for BF4 Battlelog
